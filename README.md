# start-custom
A (hopefully) easy to customize start page.  
New sections can be added like so: 
```html
<ul><li><a href="Your-link-here" class="front">yourtitlehere</a></li></ul>
<ul>
	<li><a href="Your-sublink-here">yoursublink</a></li>
	<li><a href="Your-sublink-here">yoursublink</a></li>
</ul>
```
